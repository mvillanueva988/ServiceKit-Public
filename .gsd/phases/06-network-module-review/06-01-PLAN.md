<task type="auto">
  <name>Task 1: Get-NetworkDiagnostics + wrapper async en Network.ps1</name>
  <files>modules/Network.ps1</files>
  <action>
Agregar al final de Network.ps1 dos funciones nuevas (no modificar Optimize-Network ni Start-NetworkProcess):

**`Get-NetworkDiagnostics`**:
- Leer TCP AutoTuning level con `Get-NetTCPSetting -ErrorAction SilentlyContinue | Select-Object -First 1 -ExpandProperty AutoTuningLevelLocal`; fallback: parsear `netsh int tcp show global` con regex si el cmdlet falla.
- Por cada adaptador activo (`Get-NetAdapter | Where-Object Status -eq 'Up'`): nombre, LinkSpeed, MediaType.
- DNS servers con `Get-DnsClientServerAddress -ErrorAction SilentlyContinue | Where-Object AddressFamily -eq 2` (IPv4 únicamente). Agrupar por InterfaceAlias.
- Ping latencia: `Test-Connection -ComputerName '8.8.8.8' -Count 2 -ErrorAction SilentlyContinue`. Si falla, valor -1.
- Retornar `[PSCustomObject]@{ TcpAutoTuning = string; Adapters = PSCustomObject[]; DnsServers = hashtable; PingMs = int }`

**`Start-NetworkDiagnosticsProcess`**:
- Misma estructura de serialización que `Start-NetworkProcess`: `${Function:Get-NetworkDiagnostics}.ToString()` + `Invoke-AsyncToolkitJob`.
- Nombre de job: `'NetworkDiagnostics'`.
- Sin parámetros adicionales.

Mantener `Set-StrictMode -Version Latest` al inicio del archivo (ya existe).

  </verify>

<task type="auto">
  <name>Task 2: Opción [d] Diagnósticos en networkLoop (main.ps1)</name>
  <files>main.ps1</files>
  <action>
En el sub-menú de red (`:networkLoop while ($true) {`), agregar la opción `[d]` ANTES del prompt de selección de adaptadores.

1. En el texto del menú (`Write-Host` de opciones), agregar una línea:
   ```
   Write-Host '  [d]  Diagnosticos de red  ' -NoNewline; Write-Host '(TCP, DNS, velocidad, ping)' -ForegroundColor DarkGray
   ```
   Colocarla justo antes de la línea del prompt `Read-Host`.

2. En el bloque de selección (donde se evalúa `$selection`), agregar ANTES del bloque de `'q'`:
   ```powershell
   if ($selection -eq 'd') {
       Write-Host "`n  Diagnosticando red..." -ForegroundColor Cyan
       $diagJob    = Start-NetworkDiagnosticsProcess
       $diagResult = Wait-ToolkitJobs -Jobs @($diagJob)
       [string] $tuning = if ($diagResult.TcpAutoTuning) { $diagResult.TcpAutoTuning } else { 'desconocido' }
       Write-Host ''
       Write-Host '  TCP AutoTuning : ' -NoNewline; Write-Host $tuning -ForegroundColor Cyan
       Write-Host ''
       if ($diagResult.Adapters.Count -gt 0) {
           Write-Host '  Adaptadores activos:' -ForegroundColor DarkCyan
           foreach ($a in $diagResult.Adapters) {
               Write-Host ('    {0,-28} {1}' -f $a.Name, $a.LinkSpeed)
           }
           Write-Host ''
       }
       if ($diagResult.DnsServers.Count -gt 0) {
           Write-Host '  DNS (IPv4):' -ForegroundColor DarkCyan
           foreach ($iface in $diagResult.DnsServers.Keys) {
               [string] $servers = ($diagResult.DnsServers[$iface]) -join ', '
               Write-Host ('    {0,-28} {1}' -f $iface, $servers)
           }
           Write-Host ''
       }
       if ($diagResult.PingMs -ge 0) {
           Write-Host ('  Latencia 8.8.8.8 : {0} ms' -f $diagResult.PingMs) -ForegroundColor $(if ($diagResult.PingMs -lt 50) { 'Green' } elseif ($diagResult.PingMs -lt 150) { 'Yellow' } else { 'Red' })
       } else {
           Write-Host '  Latencia 8.8.8.8 : sin respuesta' -ForegroundColor Red
       }
       Write-Host ''
       Read-Host '  [Enter] para volver' | Out-Null
       continue networkLoop
   }
   ```

El dot-source de Network.ps1 ya existe al inicio de main.ps1 (verificar y no duplicar).
